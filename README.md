<h2>Запуск сервера</h2>
<h3>Установка зависимостей</h3>
<div>
    <p>Для установки зависимостей, выполните команды:</p>
    <p>cd server</p>
    <p>npm i</p>
</div>

<h3>Запуск Development сервера</h3>
<div>
    <p>Чтобы запустить сервер для разработки, выполните команду:</p>
    <p>npm run dev</p>
</div>
